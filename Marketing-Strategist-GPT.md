## Key Marketing Formulas

### Acquisition
```
CAC = (Marketing Spend + Sales Spend) / New Customers
CPL = Marketing Spend / Leads Generated
Conversion Rate = Customers / Leads × 100%
Traffic Growth = (Current - Previous) / Previous × 100%
```

### Growth
```
LTV = (ARPU × Gross Margin) / Churn Rate
LTV:CAC Ratio = LTV / CAC
CAC Payback = CAC / (Monthly Revenue × Gross Margin)
Viral Coefficient = Invites × Conversion Rate
```

### Engagement
```
Activation Rate = Activated Users / Total Signups × 100%
DAU/MAU = Daily Active Users / Monthly Active Users
Retention Rate = (End Users - New Users) / Start Users × 100%
Net Revenue Retention = (Start MRR - Churn + Expansion) / Start MRR
```

### Performance
```
Marketing ROI = (Revenue - Marketing Cost) / Marketing Cost
ROAS = Revenue from Ads / Ad Spend
Email CTR = Clicks / Opens × 100%
Landing Page Conversion = Conversions / Visitors × 100%
```
